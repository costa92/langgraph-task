# LangGraph Task

A Python project for working with LangChain and LangGraph.

## Installation

```bash
pip install -e .
```

## Features

- Integration with LangChain
- Support for LangGraph
- Python 3.12+ compatibility

## Usage

```python
from langgraph_task.cli import main

if __name__ == "__main__":
    main()
```
